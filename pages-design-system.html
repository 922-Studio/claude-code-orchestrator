Reference page

One system, two voices.

This page mirrors the visual language used by studio.922-studio.com and gregor.922-studio.com. Both sites share typography, layout, the dot-grid + drifting-blob background, dark/light theming, and component styling. They differ only in their primary accent color — emerald for Studio, indigo for Portfolio — and the blobs that ride on it.

Background

Dot grid + drifting blobs

Two soft blobs (the accent-from + accent-to colors) drift in a blob-drift 25–30s loop, layered behind a 24px dot grid. Always z-index: 0; content sits on z-index: 10. 

.bg-pattern { background-image: radial-gradient(circle, var(--color-dot) 1px, transparent 1px); background-size: 24px 24px; }
.blob-a     { filter: blur(80px); animation: blob-drift 25s ease-in-out infinite; }
.blob-b     { filter: blur(80px); animation: blob-drift 30s ease-in-out infinite reverse; }

How to use

Mirror this on either site

  1. Apply .variant-studio or .variant-portfolio on body to lock the accent.
  2. Apply .light or .dark on html to switch theme.
  3. Use only the documented tokens — never hard-code a color outside --color-accent-*.
  4. Sections separate with .section-divider. Cards always sit on --color-surface with --color-border.
  5. For Tailwind 4 projects, the same names work as utility classes (bg-surface, text-text-muted, border-border, bg-accent-from/15).
