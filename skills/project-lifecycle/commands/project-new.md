# Bootstrap a new project

Bootstrap a whole new 922-Studio project end-to-end — GitHub repo, server infra, local
setup, monitoring, and orchestrator docs — following a proven existing project as the pattern.

Argument (optional): the project name, and/or the reference project to clone
(e.g. `/project-new anime-tracker like Anime-API`).

Load and follow the playbook, then proceed phase-by-phase with gates:

1. Read `/Users/gregor/dev/922/orchestrator/skills/project-lifecycle/ARCHETYPES.md`
2. Read `/Users/gregor/dev/922/orchestrator/skills/project-lifecycle/add.md`
3. Execute it, starting with Phase 0 preflight.

Do not skip the confirmation gates before repo creation, server mutations, or data changes.
