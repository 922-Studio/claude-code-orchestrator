Replace the legacy Indigo/cream Tailwind theme with the mono-first, teal-accent home-lab design system. Single PR: foundation + every feature page, dark default + light variant, JetBrains Mono + Inter, semantic colors only.

Locked decisions
Theme scopeDark default + light variant on the same --hl-* tokens (override block under .light root). Keep theme-toggle.
DS locationsrc/styles/design-system.css, imported once in src/main.tsx alongside index.css.
PR scopeOne mega-PR: foundation + AppShell + every feature page migrated. No half-ships.
TailwindKeep Tailwind for layout/utility; bridge @theme inline vars to the new --hl-* palette so existing utility classes (bg-card, text-muted-foreground …) point at the new tokens. No bulk class rewrite where mapping is enough.
Component classesNew .hl-* classes are first-class. Where a shared component (Button, Badge, Card) already exists, refactor its internals to emit .hl-* chrome; do not introduce a parallel system.
RadiusSwitch from the current --radius: 0rem sharp-edge to the DS's 6px default + 4px / 10px variants.
TestsUnit + E2E must stay green. Snapshot/visual-only tests get regenerated; behavior tests must not be relaxed to pass.
